# Hand Pen 🖊️

AI-powered virtual drawing using hand gestures.

## 🚀 Live Demo
https://hand-pen.vercel.app

## ✨ Features
- Draw using index finger
- Change colors with gestures
- Erase using palm
- Real-time hand tracking

## 🛠 Tech Stack
- HTML, CSS, JavaScript
- MediaPipe (Hand Tracking)

## 📌 How it works
Uses MediaPipe to detect hand landmarks and maps finger movement to a canvas.

## ⚠️ Note
Camera permission is required.
